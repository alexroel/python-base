# Control de Flujo 

~~~python
#Condicionales 
a = 0
if a > 0:
    print('Es número Positivo')
elif a == 0:
    print('El número es Neutro')
else:
    print('El número es Negativo')


#Bucle While 
while a <= 5:
    print(a)
    a += 1

#Iterador for 
mumeros = [1,2,3,4,5]
for n in mumeros:
    print(n)
~~~
